# Contributing
Thanks for your interest in contributing to HTModLoader! Please take a moment to review this document before submitting a pull request.

## Reporting Bugs
When filing an issue, make sure to answer these five questions:

* What version of HTModLoader and SDK are you using (version number)?

* What operating system and game backends are you using?

* What did you do?

* What did you expect to see?

* What did you see instead?

Use a clear and descriptive title for the issue to identify the problem.

## Making Changes
Changes are welcome via Github pull requests.
