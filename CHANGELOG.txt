This file contains the changelog of HTModLoader.


-------------------------------------------------------------------------------
 Version 1.7.0 WIP
-------------------------------------------------------------------------------

Changes:
- Fixed the incorrect display of console text when multiple '\n' exist in a single
  string.
- The main menu won't reset its size on every launch of the game.
- Fixed possible buffer overflow in HTML Console.
- Formatted all function declarations in htmodloader.h and htinternal.h
- Added HTOptionGetCustom() and HTOptionSetCustom() for mods to save their own options
  and config.
- Added demo for HTOptionGetCustom() and HTOptionSetCustom().
- Introduced LevelDB and ZLib.
- Added HTDataStore() and HTDataGet() to save persistent data.
- Added demo for HTDataStore() and HTDataGet().
- Added HTDataStoreStringKey() and HTDataGetStringKey().

-------------------------------------------------------------------------------
 Version 1.6.0 (2025.10.16)
-------------------------------------------------------------------------------

Changes:
- Added HTGetLoaderVersion() and HTGetLoaderVersionName() to get the loader's
  runtime version.
- Added HTHotkeyBindGet() to get a bound key from a key binding.
- Fixed the incorrect display of key bindings that were only read from the options
  and not explicitly registered.
- Added basic api demo for modtest.
- User can move HTML Main Menu when switched to `Console` tab.
- The console will scroll to the bottom automatically when new text is added.
- Added shared mutex for the console texts.
- Optimized HTGetModInfoFrom().
- Added HTImGuiDispatch() to expose GImGui to mods that can't link htmodloader.lib.
- Fixed incorrect display when scrolling console text when the last line is the
  longest line.
- Fixed incorrect width calculation of console text when the longest line is not visible.
- Auto-scrolling now sets the X scroll value to 0.
- Fixed a memory leak that occurred when saving options.json.


-------------------------------------------------------------------------------
 Version 1.5.0 (2025.10.12)
-------------------------------------------------------------------------------

Breaking Changes:

- Updated ImGui to v1.92.2b.
- Renamed `HTSignature` to `HTAsmSig`.
- Removed `name` field in HTAsmSig.
- Renamed `HTHookFunction` to `HTAsmFunction`.
- Added `name` field in `HTAsmFunction`, for debug use.
- Modified the order of field `fn` and field `detour` in the `HTAsmFunction`.
- Added HTAsmHookCreateRaw(), HTAsmHookCreateAPI(), HTAsmHookCreate(), 
  HTAsmHookEnable(), HTAsmHookDisable() to create and install hooks.
- Added HTTellText(), HTTellTextV(), HTTellRaw(), HTTellRawV() to print texts
  in the in-game console.
- Used thread_local to storage error codes of HTSetLastError() and HTGetLastError().

Other Changes:

- The mod scanner will skip mods with the same package name among the scanned
  mods.


-------------------------------------------------------------------------------
 Version 1.4.0-beta (2025.10.09)
-------------------------------------------------------------------------------

Breaking Changes:

- Mods will be scanned and loaded when the game is creating main window.
- Mod's exported HTModOnEnable() will be called when the game is started.

Other Changes:

- Fixed missing FindClose() of the mod scanner.
- Renamed all internal functions.


-------------------------------------------------------------------------------
 Version 1.3.0-beta (2025.09.28)
-------------------------------------------------------------------------------

Breaking Changes:

- Added HTSetLastError() and HTGetLastError() to get further info when the HTML
  APIs has failed.
- The mod loader will save the key binding data for the mods.

Other Changes:


-------------------------------------------------------------------------------
 Version 1.2.0-beta (2025.09.25)
